# AI Company OS

统一研发与经营管理平台的第一轮基础工程。

## 在线演示

GitHub Pages 部署完成后访问：<https://zyqmusic.github.io/ai-company-os/>

## 本地运行

```bash
pnpm install
pnpm dev
```

访问 `http://localhost:3000`。

## 目录

- `app/`：Next.js 管理后台页面。
- `index.html`：无需构建即可部署的静态演示页。
- `.github/workflows/deploy-pages.yml`：GitHub Pages 自动部署工作流。
